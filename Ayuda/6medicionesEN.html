6. MEASUREMENTS ON THE SCENE

To start, choose the video you want to analyze, in File → Open. Subsequently, the user must stop the video in the frame on which the annotations are to be made, and its selection is made by pressing the button Select frame, located between the two image display windows. Immediately, controls for video annotations, necessary for taking measurements, will be activated.

For taking measurements, the video must be calibrated. The calibration status indicates whether the calibration file (nameVideo) _CalibrationData.txt is found in the same folder as the video file (see Figure 13). The user can perform the calibration process at any time, as long as a video frame has been selected for analysis.

Figure 13. Calibration status (Top: calibrated video, bottom: uncalibrated video).

  • 6.1 Calibration process

    • The calibration process consists in the calculation of a homography that allows transforming a quadrilateral on the road in the video, in a rectangle of known dimensions (in meters) of the scene. The user must know what pixel positions on the video make up a quadrilateral, corresponding to a rectangle of known dimensions in the real scene.

    Video calibration steps:

    1. To start the calibration process, press the button ►, which is next to the calibration status indicator (see Figure 13). A new window with the previously selected frame will appear (see Figure 14).
    2. Write the measurement in meters of the road length distance.
    3. Write the measurement in meters of the road width distance.
    4. Press the button Start.
    5. Indicate in the image, the points corresponding to the distances.
      • Point 1: Start of the distance "road length".
      • Point 2: End of the distance "road length" and start of the distance "road width".
      • Point 3: End of the distance "road width" and start of the distance "road length".
      • Point 4: End of the distance "road length".

      Note that the "Length" measure corresponds to the distance in meters on the real scene between points 1 and 2, while the "Width" measure corresponds to the distance between points 2 and 3. When entering the last point of the quadrilateral, in the right window, the rectified image of the region defined by the user will appear, which corresponds to the rectangular area of known dimensions (see Figure 15).

    6. If the user is satisfied with the transformation, when accepting, the calibration file will be saved.

    Figure 14. Calibration process in a video.

    Figure 15. Scene rectification.

  • 6.2 Measurements of distance on the road

    • To take measurements on the road, the line tool, of the shape annotations, is used. The user must select the initial point of the line, and then the other point of the line. If annotations of rectangle and ellipse shapes exist, the ends of the line can be attached to the center of these annotations, bringing the cursor closer to the center, which will change the its color to green, indicating that the center will be selected (see Figure 16).

    In the information panel the data of the line will be recorded, presenting the distances in pixels and meters. If the video is not calibrated, the measurement in meters will be presented with a dash (-m).
