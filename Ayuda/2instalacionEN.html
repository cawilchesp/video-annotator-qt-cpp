2. INSTALLATION

Requirements:

  • Windows 7/8/8.1/10.
  • Display with a minimum resolution of 1366 × 768 pixeles.
  • Microsoft Visual C++ 2017 Redistributable (x86).

To install, run the installation program. The program will request the installation of Microsoft Visual C ++ 2017 Redistributable (x86), if you have not installed it or do not know it, you must accept its installation.

Once the installation is complete, the program will generate a shortcut in the list of applications in the start menu.
