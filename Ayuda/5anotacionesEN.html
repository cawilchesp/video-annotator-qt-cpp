5. VIDEO ANNOTATIONS

To start, choose the video you want to ANALYZE, in File → Open. Subsequently, the user must stop the video in the frame on which the annotations are to be made, and its selection is made by pressing the button Select frame, located between the two image display windows (see Figure 4). Immediately, controls for video annotations will be activated.

Figure 4. Choose frame to make annotations.

  • 5.1 Text annotations

    • The tool allows to make text annotations on a selected video frame. For these annotations, a text control section is presented (see Figure 5 ). The 6 options of the text control section are:

    1. Text font.
    2. Font size.
    3. Bold text.
    4. Italic text.
    5. Text color.
    6. Write annotation button.

    Figure 5. Options for text annotations.

    When selecting the color, the button Text color will indicate the chosen color. To start writing, select the button Write annotation, and then choose the place in the image where you want to write the annotation. A text box will appear where the annotation will be written (see Figure 6).

    Figure 6. Text annotation example.

  • 5.2 Shape annotations

    • The tool allows to make annotations using shapes on a selected video frame. For these annotations, a shape control section is presented (see Figure 7).

    Figure 7. Options for shape annotations.

    The 5 options of the shape control section are:

    1. Shape thickness.
    2. Shape color.
    3. "Rectangle" shape.
    4. "Line" shape.
    5. "Ellipse" shape.

    When selecting the color, the button Shape color will indicate the chosen color. To make an annotation of one of the three shapes on the video frame, proceed as follows:

    • Rectangle: choose one of the corners of the desired rectangle, and then choose the opposite corner (see Figure 8).
    • Line: choose one end first and then the other end is chosen.
    • Ellipse: the same procedure to draw the rectangle. The ellipse will be drawn inscribed to the selected rectangle.

    Figure 8. Rectangle-shaped annotation example.

    The shape annotations record the information in the panel located to the right of the application. The information that is registered is the following:

    Rectangle Frame number Annotation label Top left corner - Coordenate X Top left corner - Coordenate Y Bottom right corner - Coordenate X Bottom right corner - Coordenate Y Center - Coordenate X Center - Coordenate Y
    Line Frame number Annotation label Line point 1 - Coordenate X Line point 1 - Coordenate Y Line point 2 - Coordenate X Line point 2 - Coordenate Y Distance in pixels Distance in meters

  • 5.3 Zoom in, zoom out, and undo annotations

    • The Options control section, as shown in Figure 9, has:

    Figura 9. Annotation options.

    1. Zoom in.
    2. Zoom out.
    3. ROI selection (for taking samples, see Section 7).
    4. Undo.

    The option Zoom in allows you to control the zoom of the image to facilitate making annotations that require a more accurate position. The selection of the region to be zoomed in is similar to that of drawing a rectangle (see Figure 10). However, the zoomed image is presented keeping the aspect ratio (width/height) of the video. Therefore, the zoomed image may correspond to a larger area than the selected one.

    Figure 10. Zoomed in image.

    The option ROI (Region of interest), allows the selection of a specific area of the scene, useful for taking samples/examples, as presented in Section 7. SAMPLES/EXAMPLES IN VIDEO. This selection is made in the same way as drawning a rectangle. The image presented will correspond to the selected region (see Figure 11). The user will not be able to write or draw annotations on the ROI.

    Figure 11. ROI selection example in the scene.

    The option Zoom out is activated when the user has zoomed in or when a ROI was selected. This option allows you to return to the original size of the video, preserving the previously made annotations.

    The option Undo allows you to erase one by one, the text or shape annotations in the image, including the corresponding information registered in the panel.

  • 5.4 Record of annotations

    • In the section Change Frame, the user can quickly change the frame to make annotations.

    Figure 12. Options of the section Change Frame.

    This section has:

    1. Previous frame.
    2. Next frame.

    The buttons Previous Frame and Next Frame allow to load the previous or next frame to analyze, for ease to the user. In addition, when changing frames using this option, the information from the previous annotations is maintained in the panel on the right.

    This option saves the annotations of the image and panel information automatically, with the names of the files described in the next section. As annotations are made in more video frames, the files are updated and the images are added to the configured folder.

  • 5.5 Save annotations

    • When you select a video frame to make annotations, the names of the result files will appear at the top of the image and the information panel, based on the name of the video and the selected frame.

    The button Save image will save in a Portable Network Graphics (.PNG) file, the current image presented in the window, keeping the annotations present, or the ROI selected.

    The button Save text will save in both a text file (.TXT) and a JavaScript Object Notation (.JSON) file, the information registered in the information panel to the right of the application. If the panel is empty, the user will be informed and no information will be saved.
