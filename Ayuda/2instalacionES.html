2. INSTALACIÓN

Requisitos:

  • Windows 7/8/8.1/10.
  • Pantalla con una resolución mínima de 1366 × 768 pixeles.
  • Microsoft Visual C++ 2017 Redistributable (x86).

Para instalar, ejecutar el programa de instalación. El programa solicitará la instalación de Microsoft Visual C++ 2017 Redistributable (x86), si no lo ha instalado o no lo sabe, se debe aceptar su instalación.

Una vez finalizada la instalación, el programa generará un acceso directo en la lista de aplicaciones del menú inicio.
